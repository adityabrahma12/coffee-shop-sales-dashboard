# coffee-shop-sales-dashboard

This project is an end-to-end Excel Dashboard & Business Analysis project built using retail sales data from a coffee shop business.

The goal of this project was not just to create charts and dashboards, but to think like a real business analyst and answer practical business questions that can help a coffee shop improve its sales, operations, and customer experience.

This project focuses on transforming raw sales data into meaningful business insights that can support better decision-making.
